# Hermes
Sistema de detección de phishing multi-agente diseñado específicamente para el mercado español. Detecta suplantación de bancos (CaixaBank, BBVA, Santander, ING), instituciones (Hacienda/AEAT, DGT, SEPE) y paquetería (Correos), analizando emails, URLs y SMS/WhatsApp. Prioriza privacidad (RGPD) y velocidad (~14ms por análisis).
